💾 Export Backup

Downloads all your data — leads, proposals, projects, contacts, settings, and everything else — as a single JSON file. Save it to your computer or Google Drive. Do this regularly as Supabase free tier does not include automatic backups.

Recommended backup schedule:
Weekly if you're entering data regularly — monthly at minimum. The file is small and takes one click.

⬆ Restore from Backup

Select a previously exported backup file to restore your data. This will overwrite all current data in the cloud. Only use this if you need to recover from data loss.
